# Verzon

> The only release tool you'll ever need

- 🚀 **Blazingly Fast** - Built in Rust for maximum performance
- 🔄 **Automated Versioning** - Semantic versioning made simple
- 📝 **Smart Changelogs** - Automatic changelog generation
- 🔗 **CI/CD Ready** - Seamlessly integrates with your pipeline
- 🎯 **Zero Config** - Works out of the box with sensible defaults
- 🛠️ **Flexible** - Customizable to fit your workflow

Verzon is a modern release automation tool built in Rust for speed and reliability. It streamlines your versioning, changelog generation, and release workflows, allowing you to focus on building great software instead of managing releases.
